# Amazon Automation Assignment

## Tech Stack
- Java
- Playwright
- JUnit 5
- Maven

## Test Cases
1. Search iPhone, add to cart, print price
2. Search Galaxy device, add to cart, print price

## Parallel Execution
Implemented using JUnit 5 parallel execution.

## How to Run

### Option 1: IntelliJ IDEA
Open as Maven project and run AmazonParallelTest.java

### Option 2: Command Line
mvn test
(Requires Maven installed and configured in PATH)
